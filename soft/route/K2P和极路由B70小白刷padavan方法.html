本人小白一个，才入手了一台K2P和极路由B70，K2P很赞，极路由B70固件不稳定，连的人一多偶尔会自动重启。
不过我仍然喜欢老毛子的固件（信号强，系统稳定，能去广告）。因此研究了下怎么刷机。
论坛上贴子很多，我整理了一下，把自己刷机的过程记录下来，希望对各位小白有所帮助，各位老鸟可以无视，哈哈。


K2P小白刷机方法
一，开启telnet功能
1、进入http://iytc.net/k2p.php
2、输入你的WAN口MAC地址，下载配置文件(如验证码不能校验，请开启cookie或更换浏览器再试)
3、用此配置文件恢复即可开启telnet
以上参考：http://www.right.com.cn/forum/thread-217088-1-1.html
二，准备刷机
用telnet登录K2P
root@K2P:/#cd /tmp
root@K2P:/tmp#opkg install http://downloads.openwrt.org/chaos_calmer/15.05/ramips/mt7621/packages/packages/wget_1.16.3-1_ramips_1004kc.ipk
安装成功应该可以看到有wget-ssl文件了
root@K2P:/tmp# ls /overlay/usr/bin/
dbclient     dropbearkey  scp          ssh          wget-ssl
下载完成后校对一下md5是不是 56d16166de9deb781bc18b3f8b7a7ee6 ,如果OK,继续下一步操作
root@K2P:/tmp# md5sum breed-mt7621-phicomm-k2p.bin
56d16166de9deb781bc18b3f8b7a7ee6  breed-mt7621-phicomm-k2p.bin
写入breed:
root@K2P:/tmp# cat /proc/mtd
dev:    size   erasesize  name
mtd0: 01000000 00010000 "ALL"
mtd1: 00030000 00010000 "Bootloader"
mtd2: 00010000 00010000 "Config"
mtd3: 00010000 00010000 "Factory"
mtd4: 00050000 00010000 "permanent_config"
mtd5: 00f60000 00010000 "firmware"
mtd6: 00c48b96 00010000 "rootfs"
mtd7: 00200000 00010000 "rootfs_data"
root@K2P:/tmp# mtd -r write /tmp/breed-mt7621-phicomm-k2p.bin Bootloader
写入成功的情况下会自动重启的.
以上参考：http://p4davan.80x86.io/blog/2017/06/19/k2p-stock-rom-install-breed-bootloader/
重启完成之后，断电，长按复位键，不要松开，插上电源等10秒再松开。
设置电脑IP地址，进入192.168.1.1

上传下载的固件。
最新固件下载地址：http://p4davan.80x86.io/download/
重启之后就OK了。
K2P新固件的IP地址是192.168.4.1



极路由B70小白刷机方法
一，先获取SSH权限
支持以下两个版本：1.2.5.15805s， 1.3.5.18462s
B70联网，安装Aria2Gee（无内网穿透版）插件。（让WAN口能上网）


注意PATH要改成 /   下一步， 确定。
打开我的电脑， 在地址栏输入：\\hiwifi
会提示用户名和密码
用户名是admin 密码为极路由的密码。
进入这个目录：\\hiwifi\Aria2Gee\etc\market
找到Aria2GeeFree.script文件， 然后用NOTEPAD++之类的文本编辑器打开， 系统自带的记事本打开格式有问题。
然后找到uninstall(){这一行， 在下面添加：
    /etc/init.d/dropbear enable
    /etc/init.d/dropbear start
然后保存。

卸载 Aria2Gee（无内网穿透版）插件

SSH连接路由器，端口：22， 账号：root  密码是你路由器进入后台的密码
以上可参考：http://www.right.com.cn/forum/thread-220214-1-1.html
二，准备刷机
备份mac地址 弄个txt文件保存一下现有系统的mac地址,如:
wan: D6:EE:07:08:08:07lan: D4:EE:07:08:08:06

然后把路由器拔电，长按复位键，不要松开，插上电源，等10秒再松开。
设置一下电脑的IP，进入192.168.1.1
刷入固件
最新的固件可以自己到http://p4davan.80x86.io/download/ 下载
重启之后把电脑改成自动获取IP。这时候路由器的IP地址变成了192.168.5.1
mac地址修复
padavan成功启动后,做一次双清. 再次启动后, 登录进ssh或直接在控制台操作:
!! 注意,不要照这个原样复制,这个只是demo, 实际命令你要用自己备份下来的mac地址.
[B70 /home/root]# lan_eeprom_mac D4:EE:07:08:08:06
LAN EEPROM MAC address: D4:EE:07:08:08:06


Please reboot router!
[B70 /home/root]# wan_eeprom_mac D4:EE:07:08:08:07
WAN EEPROM MAC address: D6:EE:07:08:08:07
Please reboot router!
[B70 /home/root]# sync[B70 /home/root]# reboot
以上内容可参考：http://p4davan.80x86.io/blog/2017/06/24/totolink-a3004ns-and-hiwifi-b70-firmware-released/
到此，刷机完成。
